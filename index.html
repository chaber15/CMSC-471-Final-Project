Each decision tree is trained to split the data based on entropy

  • Entropy measures the "impuruty" or disorder of a dataset.
  • At each node, the tree chooses the feauture and threshold that results in the greatest information gain, or the largest reduction in entropy.
  • This process repeats until the tree reaches a maximum depth, or akk samples in a node belong to the same class.

  • Each tree independently makes a prediction based on the input features(e.g., "Home win" or "Away win").
  • The Random Forest aggregates these predicitions using majority voting:
    • If mose trees say "Home win," the overall predicition is "Home win."
